AOV网与拓扑排序

定义：

  • 以顶点表示活动，以有向边表示活动之间的优先关系的有向图叫做 AOV 网
  • 在 AOV 网中，若顶点 i 到 顶点 j 之间有有向路径，则称顶点 i 为 顶点 j 的前驱，顶点 j 是 顶点 i 的后继。若顶点 i 和 顶点 j 之间有一条有向边，那么 i 是 j 的直接前驱，j 是 i 的直接后继。

拓扑排序

​ 检查 AOV 网中是否存在回路的方法是对 AOC 网进行 拓扑排序，构成一个序列，使得该序列满足

  1. 若在 AOV 网中，顶点 i 优于 顶点 j，则在该序列中，也是顶点 i 优于 顶点 j
  2. 若在 AOV 网中，顶点 i 与 顶点 j 之间不存在优先关系，则在该序列中建立他们的优先关系，即顶点 i 优于 顶点 j 或者 顶点 j 优于 顶点 i
  3. 若能够构造出拓扑序列，则拓扑序列包含 AOV 网中的全部顶点

拓扑排序方法

  1. 从 AOV 网中任选择一个没有前驱（入度为0）的顶点
  2. 从 AOV 网中去掉该顶点以及该顶点出发的所有便
  3. 重复上述过程，直到
    • 网中所有的顶点都被去掉 ----无回路
    • 网中还有顶点，但不存在入度为 0 的顶点 ----有回路

关键路径（AOE网）

点表示瞬时时间，边表示消耗一定时间的事件

  1. 关键路径：从源点到汇点的最长路径，即工程所需要的最小完工时间

  2. 关键路径上的活动（边）为关键活动：最早开工时间（\(e_i\)） == 最晚开工时间（\(l_i\)）

  3. 顶点（事件）的最早触发时间 Ve(i) : max { Ve ( j ) + W( j , i ) } （j 是 i 的前驱顶点集合中的点）

    顶点（事件）的最晚触发时间 Vl(i) : min { Vl ( j ) - W( j , i ) } （k 是 i 的后继顶点集合中的点）

    汇点的最早触发时间等于最晚触发时间

表的基本概念

  • 查找表：一种以集合为逻辑结构，以查找为核心运算，同时包括其他运算的数据结构
  • 关键字：用于标识数据元素的数据项，简称键
  • 主关键字：唯一
  • 此关键字：标识若干
  • 查找：成功返回位置，不成功返回特殊标识
